O que é Especificidade e Seletores?

Seletores

Os seletores do CSS são responsáveis por determinar quais elementos do HTML receberão estilização, ou seja, é um padrão que determina qual recurso será estilizado na sua página ou aplicação. Em suma, Os Seletores definem quais elementos um conjunto de regras CSS se aplica.

Seletores Básicos

  • Seletor por tag

    Este seletor básico escolhe todos os elementos que correspondem ao nome fornecido.

    • Sintaxe:

    div {
    color: blue ;
    }

  • Seletor por classe

    Este seletor básico escolhe elementos baseados no valor de seu atributo classe.

    • Sintaxe:

    .first {
    color: red ;
    }

  • Seletor por ID

    Este seletor básico escolhe nós baseados no valor do atributo id. Deve existir apenas um elemento com o mesmo ID no mesmo documento.

    • Sintaxe:

    #second {
    color: green ;
    }

Especificidade

A especificação é a maneira de como os navegadores definem quais valores de propriedades são os mais relevantes para o elemento a ser utilizado. A especificação é baseada apenas nas regras impostas na composição de diferentes tipos de seletores.

Como calcular?

A espeficicação é calculada na concatenação da contagem de cada tipo de seletor. Não é um peso aplicado na expressão correspondente. No caso de igualdade de especificação, a última declaração encontrada no CSS é aplicada ao elemento. Na tabela abaixo podemos observar alguns valores do seletor "div", em cada situação, o valor da sua especificidade será diferente:
Seletores IDs Classes Elementos Especificidade
div 0 0 1 0-0-1
div # idTeste 1 0 1 1-0-1
div.Teste1 0 1 1 0-1-1
div.Teste1.Teste2.Teste3 0 3 1 0-3-1

De acordo com os valores acima, a especificidade que vai ser priorizada é a que tem o valor cujo
é "1-0-1". Porque os valores da esquerda vão ser sempre os mais priorizados no calculo de especificidade CSS, ou seja, os valores que tem a maior ordem de precedência.
